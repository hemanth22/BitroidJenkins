# BitroidJenkins
My custom Jenkins

## How to run a container in play with docker :## 

Go to link : https://labs.play-with-docker.com/

Download git repo using command : ```$git clone https://github.com/hemanth22/BitroidJenkins.git```

Change directory to BitroidJenkins

Run this command : ```$docker stack deploy jenkins -c pwdstack.yml```
